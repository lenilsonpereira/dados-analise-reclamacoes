Projeto pessoal em análise de dados, desenvolvido no google colab com o objetivo de praticar análise de dados.

Importante! Os dados utilizados neste projeto NÃO foram extraídos do site reclame aqui.
As informações são baseadas em relatos comuns e situações frequentemente mencionadas por consumidores no dia a dia, sendo utilizadas para fins educacionais e de simulação.


# Objetivo do projeto

Simular uma análise de reclamações de clientes para identificar padrões
e tipos de problemas mais recorrentes, como:
- Entregas marcadas como realizadas sem terem sido entregues
- Danos aos produtos durante a entrega


Este projeto não representa dados oficiais da Amazon.
